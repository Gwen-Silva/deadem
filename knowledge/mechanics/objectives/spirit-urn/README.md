# Spirit Urn / Soul Urn

This pilot package defines the Urn as a versioned objective mechanic, not as an
active replay fact. Current wiki documentation is recorded as secondary
evidence. Build 23916427 is not mapped to a patch, so no current Urn rule is
automatically applicable to replay 009.

Required future telemetry includes Urn state, position, carrier, deposit,
reward deltas, team economy context, and patch/build mapping. Without those
fields, the knowledge layer can only say what would be required to evaluate the
mechanic.
